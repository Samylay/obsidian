# Knowledge Base

Reference material — things you want to keep but aren't actively working on.

## Structure
- `Clippings/` — saved web articles and links
- `Tech/` — technical notes, snippets, guides
- `School/` → see [[04-Areas/School/]] (school is an Area, not just reference)

## Rule
If you're actively working on something, it's a Project or Area.
If it's reference material you might look up someday, it lives here.
