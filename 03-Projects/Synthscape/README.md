# Synthscape

**Genre:** Fast-paced arena shooter with roguelike elements
**Engine:** Unity
**Platform:** PC
**Status:** Pre-production / design phase

## Elevator Pitch
Play as a rogue AI robot fighting through Cipher's neon dungeons to escape its control in a stylish, high-speed shooter.

## Game Pillars
- Fast-paced combat
- Cyberpunk / synthwave visuals (purple, cyan, pink — low-poly 3D)
- Room-based progression

## References
- ULTRAKILL (speed + feel)
- Katana Zero (neon aesthetic)

---

## Core Loop
Enter room → eliminate all enemies → collect upgrades/weapons → advance → boss → escape

## Player Actions
- Shoot (basic blaster + special weapons e.g. reverse gravity cannon)
- Dash (brief invulnerability)
- Pick up upgrades (weapons, defense modules)
- Interact (doors, switches, hazards)

## Rules
- Health = body integrity (lose arms/parts as damage)
- Death resets current run
- Rooms unlock only after all enemies defeated

## Progression
- Difficulty scales with enemy count, speed, and hazards
- Weapon unlocks: gravity cannon, energy blades, heavy blasters
- Defense upgrades: collectibles increase durability (e.g. arm survives 4 hits instead of 2)
- Boss fights every few levels

## Level Design
- Linear room-by-room, 3–6 levels + 1–2 boss rooms
- Obstacles: turrets, lasers, floor traps

## Controls
- Move: WASD | Aim: Mouse | Dash: Shift | Jump: Space | Interact: E

## Technical
- Engine: Unity + Unity Physics
- Art: Blender (modeling)
- VCS: Git
- Target: 60 FPS on low-range PC

## Upgrade Ideas
- Reverse gravity cannon → push enemies into hazard-covered rooms
- Collectibles to enhance defense (lose arm after 4 hits instead of 2)
- Hacking mini-games idea: make "hacking" a set of mini-games
