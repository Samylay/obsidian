# Synthscape

2D solarpunk game built in Unity.

## Concept
- Player personas drive economy design — estimate money at each game stage
- Item costs calibrated to natural progression (Dragon Quest model, minus the grind)
- Genre: solarpunk, 2D

## Current Status
- Concept / pre-production

## Design Notes
- Different player personas define how different gamers interact with the game
- Economy: estimate money per persona per game stage → price items accordingly
- Balance: avoid insane grind, reflect actual level needed to beat the game
