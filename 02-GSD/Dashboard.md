# GSD Dashboard

> Capture everything. Commit to nothing until you've decided it matters.

## How this works
1. **Capture** — everything goes to [[01-Inbox/Inbox]] first, no exceptions
2. **Clarify** — is it actionable? Yes → Tasks. No → Knowledge or Archive
3. **Organize** — Active now, Someday, or Waiting on someone
4. **Reflect** — Sunday weekly review (30 min)
5. **Engage** — work from Active.md only, in priority order
