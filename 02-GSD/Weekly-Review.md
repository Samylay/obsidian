# Weekly Review Template

> Run every Sunday. Takes ~30 min. This is what keeps the system alive.

## Date: YYYY-MM-DD

---

## 1. Clear the decks (10 min)
- [ ] Process everything in [[01-Inbox/Inbox]] — move to tasks, projects, or archive
- [ ] Check email/messages for anything that needs capturing
- [ ] Clear desktop and downloads folder

## 2. Review active tasks (5 min)
- [ ] Open [[Tasks/Active]] — what got done? Archive completed items
- [ ] What didn't get done? Was it a priority mistake or time issue?
- [ ] Check [[Tasks/Waiting]] — any follow-ups needed?

## 3. Review projects (10 min)
- [ ] JECT — any stalled actions? Update [[03-Projects/JECT/LOG]]
- [ ] Synthscape — any stalled actions? Update [[03-Projects/Synthscape/LOG]]
- [ ] Triathlon — training on track? Update [[03-Projects/Triathlon/LOG]]
- [ ] School — any upcoming deadlines?

## 4. Plan next week (5 min)
- [ ] Pick 3 #p1 tasks for the week
- [ ] Add to [[Tasks/Active]]
- [ ] Check [[Tasks/Someday]] — anything ready to activate?

## 5. Reflection (optional)
**What went well this week?**

**What would I do differently?**

**One thing I want to focus on next week:**
